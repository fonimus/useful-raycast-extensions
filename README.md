# Raycast extensions

## Installation

### Private store

- Ask me for a raycast team link

### Manual

- Clone repository in `raycast-extensions`
- In raycast, type *Import Extensions*, enter, and choose `raycast-extensions`/`extensions`/`extension-folder`
- `yarn install -W`
- `yarn build`
