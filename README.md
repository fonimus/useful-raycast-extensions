# Raycast extensions

## Installation

### Private store

- Ask me for a raycast team link

### Manual

- Clone repository in `raycast-extensions`
- cd `extension-folder`
- In raycast, type *Import Extensions*, enter, and choose `raycast-extensions`/`extension-folder`
- `npm ci`
- `npm run build`
